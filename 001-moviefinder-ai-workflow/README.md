# AI 활용 기록 001 - MovieFinder 개발 과정

## 목적
AI 도구를 활용한 개발 경험을 기록하기 위해 작성한 첫 번째 노트

## 사용한 도구
- Claude Code
- Android Studio
- Kotlin

## 프로젝트
영화 검색 기능을 제공하는 Android 앱

프로젝트 링크  
https://github.com/ChooJeongHo/MovieFinder

## AI 활용 방식

MovieFinder 앱 개발 과정에서 Claude Code를 활용하여 다음과 같은 방식으로 작업을 진행했다.

1. 필요한 기능 목록을 먼저 정리
2. Claude Code에게 기능 구현 코드 생성 요청
3. 실행이 되지 않는 부분 확인
4. 수정 방향을 다시 요청
5. 생성된 코드를 검증하고 필요한 부분을 수정

## 적용된 기능 예시

- 영화 검색 기능
- API 호출 코드 생성
- 기본 화면 UI 코드 생성
- 기능 수정 및 오류 보완

## 느낀 점

AI를 활용하면 초기 코드 작성 속도는 빠르지만  
결과 코드를 검증하고 수정 요청을 반복하는 과정이 중요했다.
